### フローコントロール

*   if statements
*   if...else statements
*   nested if statements

---

### IF Statement

```python
if expression:
   statement(s)
```

---

### IF...ELIF...ELSE Statements

```python
if expression:
   statement(s)
elif:
   statement(s)
elif:
   statement(s)
else:
   statement(s)
```

---

### nested IF statements

```python
if expression1:
   statement(s)
   if expression2:
      statement(s)
   elif expression3:
      statement(s)
   else:
      statement(s)
   elif expression4:
      statement(s)
else:
   statement(s)
```

---

### ループ

```python
for iterating_var in sequence:
   statements(s)
```
